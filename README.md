# 💳 Credit Risk Prediction System

An end-to-end Machine Learning project that predicts the probability of loan default using customer financial and behavioral data.

This system simulates a real-world credit risk assessment pipeline used by banks and fintech companies.

---

## 🚀 Features

- Dual-model architecture:
  - **Model A** → Uses credit history (higher accuracy)
  - **Model B** → Works without credit history (new borrowers)
- Real-time prediction using Streamlit
- Custom threshold tuning for business use-case
- Feature engineering based on financial ratios
- Explainability (top risk drivers)
- Clean and production-ready pipeline

---

## 🧠 Problem Statement

Financial institutions need to assess whether a customer is likely to default on a loan.

This project predicts:

> **Will a customer default or not?**
